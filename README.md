## Build
You will need `raylib`, `git`, `gcc`, and `make`.

Download `box`:
```
git clone https://github.com/bluMATRIKZ/box.git && cd box
```

Compile `box`:
```
make
```

Run `box`:
```
./box
```

## Controls
- Mouse = Move camera
- W, A, S, D = Move player
- Left click = Place block
- Right click = Remove block
- Backspace = Uncapture/capture mouse
- Escape = Exit
- 1 to 4 = Change blocks
